# Developer Badges
A mod that gives badges to Geode mod developers.

## Features
- Clickable badges displayed on a user's profile and comments
- Comment coloring based on a user's badges

## Badges
- Mod Developer
- Verified Developer
- Index Staff
- Lead Developer
